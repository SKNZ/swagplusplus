Detailed Description

template<typename T>
class nsSdD::CList< T >

CList is the main class of our work, it's develop in order to be the most close to the original std::list In this idea we choose to make a double-linked list, in that way we can use bidirectional iterator and have a stl compliant CList. During the development process we choose to work in TDD (Test Driven Development) in order to have the least bugs possible.
